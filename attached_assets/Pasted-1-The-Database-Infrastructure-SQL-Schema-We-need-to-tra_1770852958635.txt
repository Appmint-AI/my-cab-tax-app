2. The "Auto-Grossing" Logic (Premium Feature)

This is the math that "reconstructs" the 1099-K. We use a 25% default platform fee (common for Uber/Lyft) but mark it as an estimate.

The Math for the Replit Agent:

"Implement the calculateGrossUp function in the backend:

JavaScript
function calculateGrossUp(netPayout) {
  const feeRate = 0.25; // 25% Platform Fee estimate
  const grossIncome = netPayout / (1 - feeRate);
  const platformFees = grossIncome - netPayout;
  return { grossIncome, platformFees };
}
In the UI, if user.subscription_tier !== 'pro', blur the 'Auto-Gross' toggle and show a 'Locked' icon with an 'Upgrade' tooltip."
